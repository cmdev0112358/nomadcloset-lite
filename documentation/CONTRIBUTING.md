# How to Contribute

We welcome contributions! This project is small and simple, making it a great place to start.

## Workflow

1.  **Find an Issue:** Look for an open issue in the "Issues" tab.
2.  **Fork the Repo:** Create your own copy of the repository.
3.  **Create a Branch:** Create a new branch for your feature or bugfix (e.g., `feat/add-item-sorting` or `fix/modal-close-bug`).
4.  **Commit Your Changes:** Make your changes and commit them with a clear message.
5.  **Push and Open a PR:** Push your branch to your fork and open a Pull Request (PR) against the `main` branch of this repository.

Please describe your change in the PR and link it to the relevant issue.
